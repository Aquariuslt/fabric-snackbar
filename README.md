## Usage

### Pre-Requirement
- node < 7.0
- docker
- docker-compose


### Installation
```bash
npm install
```

### Start

#### Start Fabric Network
```bash
cd src/artifacts
docker-compose  up
```

#### Start Fabric Snackbar Service

Now return to project root folder.

```bash
npm start
```
